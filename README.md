# Inside-the-Riksdag-A-data-driven-look-at-Swedish-parliamentary-motions

## Overview
This repository contains an exploratory data analysis conducted as part of an academic lab
assignment focused on political motion. The project examines patterns related to political
motions using structured data and basic analytical techniques.

The analysis is descriptive and exploratory and is intended solely for educational purposes.

---

## Project Objectives
- Analyze patterns and characteristics of political motions
- Apply data cleaning and preprocessing techniques
- Perform exploratory data analysis and visualization
- Interpret results within a political science and governance framework

---

## Methods
The analysis follows these main steps:

1. **Data Preparation**
   - Loading and cleaning the dataset
   - Handling missing values and formatting issues

2. **Exploratory Analysis**
   - Frequency and distribution analysis of political motions
   - Visualization of trends and patterns

3. **Interpretation**
   - Discussion of observed motion-related patterns
   - Contextual interpretation based on political process theory

---

## Tools & Technologies
- Python
- Jupyter Notebook
- pandas
- matplotlib / seaborn

---

## Key Observations
- Political motions vary across categories and contexts
- Certain types of motions appear more frequently than others
- Findings are descriptive and do not imply causation or evaluation
